Yahoo OptionChain Downloader - This package allows for the parsing and downloading of option data information from the yahoo finance website. Additionally, the package also contains a function that parses and downloads stock price information.

Type "help <filename.m>" at the command prompt for a full explanation of the inputparameters and some additional examples.
1 - yahooOptionDataParser.m - Parse option data information from the yahoo finance website for a given array of inputsymbols.
2 - yahooStockPriceParser.m - Parse (minimal) stock price information from the yahoo finance website for a given array of inputsymbols.
